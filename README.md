## 📌 About This Project

This repository contains my studies and implementations using the **Gameplay Ability System (GAS)** from **Unreal Engine 5**.

The goal of this project is to understand and practice one of Unreal Engine's most powerful gameplay frameworks, commonly used in large-scale productions to create scalable and complex gameplay systems.

Through this project, I am exploring how professional game systems are structured, focusing on gameplay programming using **C++** and Unreal Engine's native architecture.

# 📚 Features & Concepts Studied

## ⚔️ Gameplay Abilities

Implementation and exploration of Unreal's Gameplay Ability framework:

- Creating custom abilities;
- Ability activation and cancellation;
- Ability costs;
- Cooldowns;
- Gameplay actions and interactions.

---

## 📊 Attribute System

Creating and managing gameplay attributes:

Examples:

- Health;
- Mana;
- Stamina;
- Damage;
- Defense;
- Movement Speed.

---

## ✨ Gameplay Effects

Working with gameplay effects to modify gameplay states:

 

- Attribute modifications;
- Temporary effects;
- Buffs and debuffs;
- Damage over time;
- Status effects.
